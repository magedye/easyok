هذه نسخة جاهزة من واجهة المستخدم (Frontend) مبنية باستخدام React و TypeScript، متوافقة مع التصميم المعماري الذي سبق تقديمه. الواجهة تغطي الوظائف الأساسية: إدخال الأسئلة وبث النتائج بثلاث مراحل (بيانات، رسم بياني، ملخص)، وإدارة بيانات التدريب للمسؤول، بالإضافة إلى صفحة تسجيل بسيطة لحفظ توكن JWT. الواجهة تعتمد على بيئة Vite للتطوير السريع ويمكن تشغيلها محليًا بعد تثبيت الحزم.

**أبرز الميزات:**

* **تطبيق أحادي الصفحة (SPA)** يستخدم React Router للتنقل بين صفحات Ask/Training/Login.
* **بث النتائج عبر NDJSON**: يتم استخدام fetch لقراءة تدفق NDJSON من الخادم وتحديث الواجهة في الوقت الفعلي.
* **عرض البيانات** في جدول ديناميكي استنادًا إلى المفاتيح في الصف الأول.
* **رسم بياني تلقائي** باستخدام Chart.js يستخلص المحاور من التكوين المرسل من الخادم ويدعم الأنواع bar/line/pie.
* **ملخص نصي** يظهر المرحلة الثالثة من الاستجابة.
* **نموذج رفع بيانات التدريب** يسمح بإضافة السؤال وSQL والبيانات الوصفية بصيغة JSON.
* **شريط ملاحة** مع رابط لتسجيل الدخول/الخروج ويعرض صفحات التدريب للمستخدم بعد تسجيل الدخول.
* **إدارة التوكن** عبر localStorage، مع إمكانية تسجيل الدخول بإدخال JWT يدويًا.
* **إعدادات قابلة للتغيير** عبر متغير بيئة `VITE_API_BASE_URL` لتحديد عنوان الخادم بسهولة.

للبدء:

1. قم بفك ضغط الملف المرفق.
2. داخل مجلد `frontend/`، ثبت الحزم باستخدام:

   ```bash
   npm install
   ```
3. ثم شغّل التطبيق محليًا:

   ```bash
   npm run dev
   ```
4. تأكد من وضع قيمة صحيحة لـ `VITE_API_BASE_URL` في ملف `.env` داخل مجلد المشروع إن لزم الأمر.
